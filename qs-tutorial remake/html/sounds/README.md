# Archivos de Audio para QS-Tutorial

## Instrucciones

Coloca los siguientes archivos de audio en esta carpeta:

- `scene1.mp3` - Audio para la escena 1 (Bienvenida/Introducción)
- `scene2.mp3` - Audio para la escena 2
- `scene3.mp3` - Audio para la escena 3
- `scene4.mp3` - Audio para la escena 4
- `scene5.mp3` - Audio para la escena 5
- `scene6.mp3` - Audio para la escena 6 (Final/Despedida)

## Especificaciones Recomendadas

- **Formato**: MP3
- **Bitrate**: 128-192 kbps (para optimizar tamaño sin perder calidad)
- **Frecuencia**: 44.1 kHz
- **Canales**: Estéreo o Mono
- **Duración**: Según necesidad de cada escena

## Notas

- El sistema de audio está configurado con volumen por defecto de 0.6 (60%)
- Los archivos se cargan automáticamente cuando cambia la escena
- Si un archivo no existe, el tutorial continuará funcionando sin audio
- Los nombres de archivo DEBEN coincidir exactamente con los listados arriba
